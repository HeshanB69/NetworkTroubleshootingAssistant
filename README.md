# Network Troubleshooting Assistant

A web-based network diagnostics application built with **Python** and **Flask** to help users monitor, analyze, and troubleshoot network connectivity issues.

---

## Features

- Internet Connectivity Check
- Local IP Address Detection
- Public IP Address Detection
- DNS Resolution Test
- Gateway Detection
- Gateway Reachability Test
- Internet Speed Test
- Wi-Fi Information
- Device Discovery
- Hostname Detection
- Device Manufacturer Detection
- Port Scanner
- Automatic Troubleshooting Recommendations

## Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/NetworkTroubleshootingAssistant.git
```

Go to the project folder

```bash
cd NetworkTroubleshootingAssistant
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```

Open your browser

```
http://127.0.0.1:5000
```
